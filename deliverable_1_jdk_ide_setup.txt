
Deliverable 1: Creating the New Project with JDK & IDE Setup

1. Install JDK:
   - Download the latest JDK from Oracle or OpenJDK official websites.
   - Install it and set the JAVA_HOME and PATH environment variables.

2. Install an IDE:
   - Use IntelliJ IDEA, Eclipse, or NetBeans for efficient Java development.
   - Create a new Java project in your chosen IDE.
